README

Multi User Java Swing project which allows users to view the interior of UWO campus buildings and create their
own Points of Interest. Built along side 4 other developers. 

Dependencies Require for this running the application, fetching the local weather:
JAR Files
https://jar-download.com/artifacts/org.apache.httpcomponents/httpclient
https://jar-download.com/artifacts/com.github.cliftonlabs/json-simple/2.3.0/source-code
https://jar-download.com/artifacts/org.json

JSON File: provided in project
userData.json

To use the program run the main method in the Main.java class
